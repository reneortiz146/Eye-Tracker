Que es Eye-Tracker?

Eye-Tracker.js es una biblioteca de JavaScript que permite rastrear los movimientos oculares de un usuario a traves de una camara web y registrar la posicion de sus ojos en la pantalla. Utiliza tecnicas de vision por computadora para detectar la posicion de los ojos y seguir su movimiento mientras el usuario mira la pantalla.
